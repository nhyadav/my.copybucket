About ClipinVault

A clipboard manager built from daily frustration with lost copies.

The Story

Why I Built This

I'm Narayan Yadav, a developer based in India. I built ClipinVault because I got tired of losing links, code snippets, and terminal outputs I'd copied just minutes earlier. Every existing clipboard tool I tried either needed a cloud subscription, looked terrible, or quietly shipped my data somewhere I didn't agree to.

So I wrote my own. ClipinVault runs entirely on your machine — no accounts, no servers, no tracking. It uses SQLite for storage and AES-256 for encryption when you need it. I use it every day myself, and I keep improving it based on what I actually need.
