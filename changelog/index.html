Changelog

What's new, what's fixed, and what's next for ClipinVault.

v1.3.1 March 25, 2026 Latest
  1. Added tag-based content types: URL, Code, JSON, SQL, Command, MFA, Secrets, and Image clips alongside regular text.
  2. Clipboard now captures formatting — pastes retain the original structure of copied content.
  3. New preview button lets you view the full content of any clip before pasting.
  4. Paste options: choose to paste with or without formatting depending on the target.
  5. Image clips now support download and export — save captured screenshots directly from the vault.
  6. Redesigned icon and theme for a more professional, polished look.
  7. Full space utilization with a row-based layout for better readability across screen sizes.
  8. Auto-clean scheduler: unused clips with zero usage are cleared automatically to keep ClipinVault stable and lean.
Download for Windows Download for Ubuntu
v1.2.3 February 28, 2026 Previous Release
  1. Change content type from text to secret to encrypt data — use this only for credentials. Encryption key is stored in the Windows Key Manager or equivalent on other OS.
  2. Pin content to keep it visible on top of the list at all times.
  3. Filter and search are now available across all clips.
  4. Set a title for any content to remember context, such as which account's password was copied.
  5. Paste any content anywhere with a single click.
  6. Supports three content types without formatting: Text, Secret, and Image. (Note: v1.3.1 adds URL, Code, JSON, SQL, Command, and MFA types.)
  7. Captures screenshots and image copies — supports only text and image types for copy-paste.
