Ubuntu Installation

Follow these steps to install ClipinVault on your Ubuntu or Debian-based system.

Option 1: Via Terminal (Recommended)

Using the terminal ensures all necessary dependencies are automatically downloaded and configured.

# 1. Update your package lists $sudo apt update

# 2. Install using apt (handles all dependencies) $sudo apt install ./clipinvault_1.3.1_amd64.deb

Option 2: Via App Installer

For a purely visual approach, you can use the built-in Ubuntu Software Center or GDebi.

  • Open your Downloads folder in Files.
  • Right-click the .deb file and select Open with Software Install.
  • Click the Install button and enter your password.
  • Wait for the installation to complete and launch from your app menu.
